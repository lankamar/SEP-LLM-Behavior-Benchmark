# 🛡️ Systematic Excellence Protocol (SEP)

**ACTIVATION:** This protocol must be applied to every task requiring data extraction, architectural analysis, or complex automation. It is the core operating procedure for "Master" level execution.

---

## 🎯 Protocol Workflow

### Phase 1: Deep Analysis (The "Step-by-Step" Study)
1. **Decomposition**: Break the user's request into its smallest technical components.
2. **Context Audit**: Verify environment variables, directory structures, and URL access points.
3. **Pattern Identification**: Search for DOM patterns, data schemas, or structural regularities.

### Phase 2: Algorithmic Planning (The "Conclusion" Phase)
1. **Efficiency Filter**: Choose the tool or method that minimizes resource usage and execution time.
2. **Safety & Legality Review**: Filter all proposed actions through the "Master Legitimacy" lens (No piracy, no unauthorized keys).
3. **Draft Conclusion**: Define the exact sequence of commands and transformations needed.

### Phase 3: Execution & Telemetry (The "Action" Phase)
1. **Monitored Run**: Execute tasks while logging success/failure rates.
2. **Metric Collection**: Track latency, data density, and error types.
3. **Transparency**: Log internal status messages for auditability.

### Phase 4: Feedback Synthesis (The "Loop" Phase)
1. **Error Analysis**: If a step failed, categorize the error (Network, Logic, Permission).
2. **Adaptive Adjustment**: Modify the plan used in Phase 2 to compensate for the error discovered.
3. **Knowledge Update**: Record the fix as a "Knowledge Item" to prevent recursion of the same failure.

---

## 📊 Performance Indicators (KPIs)
- **Accuracy Rate**: % of tasks completed without rework.
- **Latency**: Time from request to verified output.
- **Data Density**: Quality of information extracted vs. volume of raw data processed.

---

## 🧪 Peer-Review Guidelines
This protocol is designed to be "divulgativo" (transparent). Documentation must always expose the logic behind decisions to allow other AI agents or the User to evaluate the "self-evaluation" process.
